    ## 1. アプリ概要
    「ふむ」は、日々の言語化できないモヤモヤを、AIとの対話を通じて自己理解という資産に変換するメンタルケアアプリです。

    - **コンセプト:** パーソナライズされた記録を簡単にストックし、モヤモヤを自己理解の資産に変える。
    - - **ターゲット:** 日常的な不安やストレスを抱えつつも、従来のジャーナリング（日記）が継続できない層。
      - - **コアバリュー:** 「入力の精神的ハードルの低減」と「納得感のあるフィードバック（新しい気づき）」。
       
        - ---

        ## 2. 機能要件 (MVP)
        ### ① AI対話型ジャーナリング機能
        - ユーザーが現在の「モヤモヤ」を自由入力または音声入力（想定）で送信。
        - - AIが傾聴しつつ、ユーザーの深層心理を引き出すような深掘り質問を返す。
          - - 1回のセッションは3〜5往復程度の対話で完結。
           
            - ### ② 自己理解分析（フィードバック）
            - - 対話終了後、AIが内容を分析し、「新しい気づき（インサイト）」を提示。
              - - 感情のラベリング（今、どんな感情だったのか）と、その背景にある価値観を抽出。
               
                - ### ③ 記録のストック（資産化）
                - - 分析結果をカード形式で保存。
                  - - 過去の記録をカレンダーやリストで振り返る機能。
                    - - 自分の感情の傾向（例：仕事に関する悩みが多い等）を可視化。
                     
                      - ### ④ 感情パレット（初期入力）
                      - - セッション開始前に、現在の気分をスタンプや色で直感的に選択。
                       
                        - ---

                        ## 3. 技術スタック案 (Claude Code向け)
                        - **Framework:** Next.js (App Router), TypeScript
                        - - **Styling:** Tailwind CSS, Shadcn/ui
                          - - **Icons:** Lucide React
                            - - **Backend/Database:** Supabase (Auth, PostgreSQL)
                              - - **AI Integration:** OpenAI API または Anthropic Claude API
                                - - **State Management:** TanStack Query (React Query)
                                 
                                  - ---

                                  ## 4. データ構造 (OOUIに基づく)
                                  ### オブジェクト: `Journal` (記録)
                                  - `id`: UUID
                                  - - `user_id`: UUID
                                    - - `mood_initial`: String (開始時の気分)
                                      - - `content_raw`: Text (対話の全ログ)
                                        - - `insight_title`: String (AIが生成した気づきのタイトル)
                                          - - `insight_body`: Text (AIによる詳細分析)
                                            - - `tags`: Array (抽出された感情タグ)
                                              - - `created_at`: Timestamp
